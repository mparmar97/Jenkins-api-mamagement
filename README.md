Developed a proof-of-concept backend application using Node.js and Express.js to automate Jenkins job management through API-based integration. The project connected a Node.js service with a Jenkins server and provided API endpoints to create Jenkins jobs, update job configuration files, trigger builds, and retrieve build execution details.

The application used Jenkins XML configuration files to define job settings and dynamically updated the XML before applying it to Jenkins. The backend used the Node.js jenkins package to communicate with Jenkins, while Express.js routes were created to expose job creation and build trigger operations. This POC helped validate how Jenkins CI/CD operations could be automated programmatically instead of being performed manually through the Jenkins UI.
